# SkillSet AI - Ultra-Modern Landing Page

A visually stunning, white-themed landing page for SkillSet AI, an AI-powered platform that helps people discover, learn, and master the right skills for their careers.

## Features

- **Modern Design**: Clean white theme with high-contrast black text
- **Animated Dither Background**: Continuous Three.js-powered background animation
- **Glassmorphism**: Beautiful glass-effect UI elements throughout
- **Smooth Animations**: Framer Motion for buttery-smooth transitions
- **Custom Blob Cursor**: Gradient cursor with soft glow effect
- **Responsive**: Fully responsive design for all devices
- **Accessibility**: Reduce motion toggle and high contrast support

## Tech Stack

- **React** - UI library
- **Vite** - Build tool
- **TailwindCSS** - Styling
- **Framer Motion** - Animations
- **Three.js** - 3D background effects

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

4. Preview production build:
   ```bash
   npm run preview
   ```
